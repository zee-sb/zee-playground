# PULL PLT deck — content for slides 15–18

Simplified: one-line value statement + milestones. Drafted from the live Jira initiatives as of **2026-06-09**.

> Note: the deck labels all four "IN PROGRESS". Live Jira phases differ — Conversation 2.0 is in Delivery, Actionable Analytics in Solution Discovery, Concierge in Problem Discovery, and Assistants 2.0 is only just Open.

---

## Slide 15 — NAVIGATOR CONCIERGE

**VALUE**
Navigator configures itself from the customer's existing content on day one — zero admin setup — unlocking the dormant tenant base and the 70+ activation waitlist.

**MILESTONES**
→ Automatic Context — the activation scan (company, audiences, topics, glossary)
→ Navigator-in-Studio — test-as-you-configure, impersonate user or group
→ Granular Controls V2 — admins scope out noisy/stale content without restructuring
→ Ships 2026.3 (July 30) as Beta / Early Access via design-partner slug list

---

## Slide 16 — CONVERSATION 2.0

**VALUE**
The conversation that doesn't drop context or break on mobile — fixing the experience layer every other bet stands on, and the drop-off that keeps return-rate flat.

**MILESTONES**
✓ Live voice (VoiceLive) and mem0 dropped — removes the demo-breaking mobile bug
→ New mobile/desktop UX — in Delivery
→ Conversation continuity — history view + restore
→ Inline source citing (no link dumps), LLM-based follow-ups, rich card UI
→ Stable transport — no mid-session drops

---

## Slide 17 — ASSISTANTS 2.0

**VALUE**
From one catch-all assistant to 50+ targeted Experts — scoped to the right content and the right groups — so Navigator becomes useful for every job, not one chat box answering for all.

**MILESTONES**
→ Use-case templates — pre-built Experts (HR, IT helpdesk, Benefits, Onboarding)
→ Smart setup — AI-suggested intranet content + starter instructions per Expert
→ Group targeting — many Experts, each scoped to a specific audience (e.g. German managers in Product)
→ Builds on shipped Assistants v1; folds into Concierge v2 as the starting set

---

## Slide 18 — ACTIONABLE ANALYTICS

**VALUE**
Navigator tells admins what to fix, not just what happened — and Evals 2.0 makes the quality numbers trustworthy, the gate every other bet depends on.

**MILESTONES**
✓ Conversation logs, evaluation v1 + AI testing pipeline — Done
→ Aggregated analytics charts — resolution rate, topics, satisfaction per assistant / audience / time
→ Evals 2.0 — replace unreliable LLM-as-judge with accurate measurement
→ Reported feedback + quality signals, each turned into a suggested next action
