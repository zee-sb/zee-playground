# Navigator Knowledge Base — How to Use This Pack

**Purpose:** these files are the curated knowledge base for the **Navigator** Claude Project. Upload all `0X_*.md` files into the Project's knowledge so the Project can answer essentially any question about Navigator — strategy, roadmap, PRDs, execution, team, and customer feedback.

**Assembled:** 2026-06-09 · Owner: Zee (Zyad Abuzeid) · Source of truth lives in Confluence (AW space) + Jira (AIW / PI initiatives) + Slack `#feedback-navigator`.

## What to do with these files

1. Open the **Navigator** Project in the Claude app.
2. Add the files below to **Project knowledge** (drag-and-drop or "Add content").
3. Paste `01_project-instructions.md` into the Project's **custom instructions** field (not as a knowledge file — it's the system prompt).
4. Re-upload the refreshable files (`04`, `05`, `07`) periodically, or ask me to regenerate them.

## A note on freshness

This pack is a **point-in-time snapshot**. For anything that changes daily — exact epic statuses, open bugs, this week's feedback — the live sources win. Each file names its live source so the Project knows where to send you for the current state.
